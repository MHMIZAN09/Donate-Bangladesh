Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) represents the structure of an HTML document in the form of a tree, where each node corresponds to an element in the document.

How do you select on element from the DOM?

To select an element from the DOM

  • getElementById
  • querySelector
  • getElementsByClassName
  • getElementsByTagName
  • querySelectorAll
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for its child elements. Instead of attaching separate event listeners to each child, the parent element listens for events that bubble up from the children.

This works because of event bubbling, where an event triggered on a child element propagates up to its parent, allowing the parent to "catch" the event.
